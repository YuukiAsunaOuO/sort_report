# 排序演算法分析報告

## 一、排序法介紹
### 1. 冒泡排序 (Bubble Sort)
原理：重複走訪要排序的數列，一次比較兩個元素，如果他們的順序錯誤就把他們交換過來。
### 2. 快速排序 (Quick Sort)
原理：選定一個基準值 (Pivot)，將數列分成比基準小和比基準大的兩部分，再遞迴排序。

## 二、複雜度分析
| 演算法 | 平均時間複雜度 | 空間複雜度 | 穩定性 |
| :--- | :--- | :--- | :--- |
| Bubble Sort | $O(n^2)$ | $O(1)$ | 穩定 |
| Quick Sort | $O(n \log n)$ | $O(\log n)$ | 不穩定 |
